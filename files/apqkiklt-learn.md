# learn-flow-Curso Redaccion

## Modulo 1

- Modelo básico de comunicacion interpersonal.
- Se debe entender la idea central del texto se debe entender, para un persona no técnica.
- Respetar partes de un informe: Introduccion-cuerpo-conclusion
- Un texto es una unidad comunicativa fundamental. Un texto se empieza con una estructura, es de caracter social, no es para mi, es para otros. Los clientes quieren informe corto-preciso-conciso.
  - Las conclusiones debe quedar como el agua.
  - Sujeto- Predicado, el sujeto debe estar explicito.
- Estructura oracional básica : Sujeto - Predicado.
- La sangría indica el inicio de una parrafo.
- Redaccion técnica.
  1.  Utilización de la estructura básica de la oración.
  2.  Utilizacion de sinónimos.
  3.  Eliminación de repeticiones léxicas.
  - Repeticion completa de una palabra tal cual otra anterior.
  - Repeticion parcial de una palabra anterior (ej: este y esta)
  1.  Eliminacion de oscuridad y ambiguedad léxica.
  - ambiguedad se da para doble interpretaciones.
  - Oscuridad se refiere a que es como un plato de tallarines y no se entiende nada.
  - Si siento que pasa esto, debe reescribir.
  1.  Eliminación de la cacofoníua.
  - Estoy contentito, porque me comeré un completito. (repetición ito)
  1.  Utilizarlas marcas notaciones del texto de forma adecuada.
  - Las mayusculas llevan tilde.
  - Las mayusculas no son para destacar. Las palabras que van con mayusculas son los nombres propios. Usar negritas, cursivas y subrayado.
  1.  Eliminación de faltas ortográficas y erratas.
  2.  Utilización de párrafo analizante.
- Tener conciencia de como se escribe.
- ¿Que es lo que me cuesta más?
  - Me cuesta organizar las ideas.
  - No tengo clara la estructura.
  - Me cuesta limitarme al espacio.

### Principios de comunicabilidad

- Reglas ortográficas -> faltas ortografica y de puntuacion.
- Cohesion -> como uno las ideas
- Coherencia -> mantenimiento de las ideas. Todo lo que se hable debe estar relacionado.
